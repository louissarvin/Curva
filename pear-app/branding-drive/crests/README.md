# Crests

These SVGs are lightweight demo placeholders inspired by national flags. They are not official FIFA marks, not official national team badges, and not affiliated with any football association. They exist so the Curva demo can render a distinct crest per team without shipping licensed artwork.

For a production release, replace each file with the licensed team crest (same filename, same viewBox).

Codes follow ISO 3166-1 alpha-3 lowercase (ita, arg, mex, zaf, esp, bra, fra, gbr).
